# Pomify - Instrucciones para Claude

## Gestor de paquetes
- Usar siempre **pnpm** en lugar de npm para instalar dependencias
- Comando de instalación: `pnpm install`
- Nunca usar `npm install` salvo que se indique explícitamente

## Stack
- Frontend: React + Vite
- Backend: Python (Flask)
- Base de datos: migraciones en `/migrations`
